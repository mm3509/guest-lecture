# Guest lecture by Miguel Morin

This guest lecture gives you a taste of the course "Computing for Business Research", B9122, where I teach software engineering and data analysis.

To follow the materials:

1. Download this repository (button "Download ZIP" at the top right) or "clone" it from a shell with `git clone https://github.com/mm3509/guest-lecture`.

2. Install the requirements with `pip install -r requirements.txt`

3. Run each file with `python mnist_ocr.py` (for example)
